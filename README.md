# Insurance Management System

A web-based platform to manage insurance policies, claims, premiums and customers.

## Tech Stack
- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express.js
- Database: PostgreSQL + Prisma ORM
- Auth: JWT + bcrypt

## Modules
- Customer Management
- Policy Management
- Claim Management
- Premium Tracking
- Document Management
- Reports Dashboard
